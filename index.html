2017-03-13

hexo部署

Hexo是一个基于Node.js的快速、简单、强大的博客框架，利用hexo可生成静态博客并托管到github上。

准备工作：

1.安装git、node.js，注意要按照自己系统来选择32位/64位版本。

2.申请github账号，创建新仓库，仓库名：你的用户名.github.io，注册后会有邮箱验证，注意查收。

安装Hexo：

1.新建文件夹（比如E:\blog）作为博客的根目录，在git bash下进入该文件夹。

2.npm install -g hexo-cli，如果官方仓库下载速度很慢，可更换国内的npm仓库：npm config set registry https://registry.npm.taobao.org。安装过程中可能会出现WARN，忽略。

3.初始化hexo：hexo init，一定要在第1步新建的文件夹路径下。

4.安装生成器：npm install，同样可能会有WARN，无视即可。

5.初步部署完成，可在本地运行博客：hexo s -g，在浏览器localhost:4000地址访问。

配置博客：

1.配置git个人信息：

git config –global user.name “你的git用户名”
git config –global user.email “邮箱地址”

如果先前已经配置好了，跳过此步。

2.用你的邮箱地址生成SSH密钥：

ssh-keygen -t rsa -C “gdutxiaoxu@163.com”

在C:\Users\Administrator.ssh下生成两个文件：id_rsa和id_rsa.pub。

3.将密钥添加到ssh-agent：

eval “$(ssh-agent -s)”
ssh-add ~/.ssh/id_rsa

4.打开id_rsa.pub，将文件里的内容复制到到github的ssh keys中。

5.输入：

ssh -T git@github.com

测试是否成功认证。

6.安装git插件：

npm install hexo-deployer-git –save

7.用notepad++或者Sublime_Text修改E:\blog中配置文件blog/_config.yml,简单配置如下：

title: Julysandz的博客 #博客的名称
subtitle: 好记性不如烂笔头 #博客副标题
description:
author: Julysandz #博客作者
language: zh-CN #语言
timezone:
…
theme: landscape #博客主题名称
…
deploy: #发布设置
type: git
repo: git@github.com:github账号名/github账号名.github.io.git #前面新建的github仓库地址
branch: master

8.部署到github：

hexo clean #清除博客缓存，避免网页出现bug
hexo g #生成静态网页到public文件夹下
hexo d #部署

部署成功后在浏览器打开：你的github用户名.github.io,就能看到你的博客了！

日常操作

1.创建文章：

hexo new “文章名称”

title: #文章标题
date: #时间，自动生成
tags: #标签
categories： #分类
