文件读取与保存

1.文本文件
val input = sc.textFile(“filepath”,partition)
读取一个文件，读取目录下所有文件，使用通配符读取特定文件

2.jason文件
作为文本按行读取，按jason格式解析
利用jason库

3.CSV
每行固定字段，逗号隔开

4.SequenceFile
Hadoop Writable类没有实现serializable接口，需要map()操作转换
val data = sc.sequenceFile(inFile,classOf[Text],classOf[IntWritable]).map{f(x)}

编程进阶

全局共享变量
1.累加器
对作业执行过程中的事件进行计数
sc.accumulator[Int]
在闭包中使用+=增加累加器值；
调用累加器的value属性访问其值。

2.广播变量
程序高效地向所有工作节点发送一个较大的只读值（比如特征向量、只读查询表）
sc.broadcast[T]
spark自动把闭包中引用的变量发送到工作节点；
一般spark会为每个操作都发送一次，通信的代价大；
广播变量只会被发送到各节点一次。
